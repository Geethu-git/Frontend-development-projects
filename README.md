# Flutter Mobile Applications Collection

A collection of Android mobile applications developed using **Flutter** in **Android Studio**. This repository showcases beginner-to-intermediate level projects focused on mobile application development, user interface design, state management, and database integration.

The applications were developed as part of academic learning and personal skill development to gain practical experience in building real-world mobile solutions.

---

##  Projects Included

###  Calculator App

A simple and responsive calculator application designed to perform basic arithmetic operations.

**Features**

* Addition, Subtraction, Multiplication, and Division
* User-friendly interface
* Real-time calculations
* Responsive design

---

###  Expense Tracker App

An application that helps users manage and monitor daily expenses efficiently.

**Features**

* Add and manage expenses
* Categorize transactions
* Track spending history
* Simple and intuitive dashboard
* Database integration for data storage

---

###  Task Manager App

A productivity application that allows users to organize and manage their daily tasks.

**Features**

* Create, update, and delete tasks
* Mark tasks as completed
* Organize daily activities
* User-friendly task management interface
* Persistent data storage

##Installation

### Clone the Repository

```bash
git clone https://github.com/Geethu-git/Web_development.git
```

###Navigate to the Project Folder

```bash
cd Web_development
```

###Install Dependencies

```bash
flutter pub get
```

###Run the Application

```bash
flutter run
```
